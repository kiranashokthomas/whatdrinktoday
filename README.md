# WhatDrinkToday

> **Let fate decide what you drink today.**

![HTML](https://img.shields.io/badge/HTML-CSS-orange) ![Status](https://img.shields.io/badge/Status-Live-brightgreen) ![Type](https://img.shields.io/badge/Type-Side%20Project-lightgrey)

A lightweight, no-framework drink randomiser web app. Open it, click the button, and it picks your drink for the day. That's it.

**Live:** [kiranashokthomas.github.io/whatdrinktoday](https://kiranashokthomas.github.io/whatdrinktoday/)

---

## What it does

- Randomises a drink suggestion when you visit or click
- Styled with a dark, minimal aesthetic using CSS custom properties
- Fully client-side — no backend, no dependencies, no tracking
- Deployed via GitHub Pages

---

## Built with

- HTML
- CSS (custom properties, responsive layout)
- Vanilla JavaScript

---

## Why

A quick side project to practice front-end layout and deployment. Sometimes you just want to build something small and ship it.
